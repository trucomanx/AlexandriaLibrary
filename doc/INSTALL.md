# alexandria-library

Program for the management and organization of digital files.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/alexandria-library/

```bash
pip install --upgrade alexandria-library
```

Using:

```bash
alexandria-library
```

## Install from source
Installing `alexandria-library` program

```bash
git clone https://github.com/trucomanx/{REPOSITORY_NAME}.git
cd {REPOSITORY_NAME}
pip install -r requirements.txt
cd src
python -m build
pip install dist/alexandria_library-*.tar.gz
```
Using:

```bash
alexandria-library
```

## Uninstall

```bash
pip uninstall alexandria_library
```
